## Program

The image consists of a ***simple program*** <br>
that uses redefined characters to create <br>
`1x1` checkerboard / stripe patterns.

<br>
<br>

## PAL / NTSC

For machines of this type, you require <br>
***proper filtering*** on the `RF Modulator`.
